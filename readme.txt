This is a classic editor extension plug-in for TCD users. It is currently offered as a beta board.

== Description ==

This plugin allows you to add your own quick tags to the classic editor.

## Third Party Services

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type TCD Classic Editor”.
3. Select the TCD Classic Editor” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
